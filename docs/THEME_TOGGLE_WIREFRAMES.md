## 3. Component State Specifications

### Theme Toggle Button States

#### State 1: Default (Not Hovered)

**Dark Mode:**
```css
background: rgba(30, 41, 59, 0.5);
border: 2px solid rgba(51, 65, 85, 0.5);
icon-color: #F59E0B (amber-500);
opacity: 1.0;
transform: scale(1.0);
```

**Light Mode:**
```css
background: #F8FAFC (slate-50);
border: 2px solid #E2E8F0 (slate-200);
icon-color: #6366F1 (indigo-600);
opacity: 1.0;
transform: scale(1.0);
```

#### State 2: Hover

**Visual Changes:**
```
Background: Slightly lighter/darker
Border: More prominent color
Icon: Rotate 180deg (Sun) or 12deg (Moon)
Transform: scale(1.1)
Shadow: Subtle glow
Cursor: pointer
```

**Dark Mode Hover:**
```css
background: rgba(51, 65, 85, 0.5);
border: 2px solid rgba(71, 85, 105, 0.5);
icon-color: #FCD34D (amber-300);
box-shadow: 0 0 12px rgba(245, 158, 11, 0.2);
```

**Light Mode Hover:**
```css
background: #E2E8F0 (slate-200);
border: 2px solid #CBD5E1 (slate-300);
icon-color: #4F46E5 (indigo-700);
box-shadow: 0 0 12px rgba(99, 102, 241, 0.15);
```

#### State 3: Focus (Keyboard)

**Accessibility Focus Ring:**
```css
outline: 2px solid #6366F1 (indigo-500);
outline-offset: 2px;
border: 2px solid #6366F1;
```
